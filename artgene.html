GENERATIVE ART

Processing code

Generative what?

"Generative Art refers to any art practice where the artist uses a system, such as a set of natural language rules a computer program, a machine, or other procedural invention, which is then set into motion with some degree of autonomy to or resulting in a complex work of art." (Philip Galanter)

What did I do?

This project was made to be in a Fanzine with a piece of generative art that was made by each student. I decided to create something with letters. In this program, strings of letters are created and repeated few times before the string changes. This happened a few times before the layer changes, creating shallowness for the end picture.
